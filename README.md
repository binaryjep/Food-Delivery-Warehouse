
# Food Delivery Data Warehouse

This repository contains a complete end-to-end data warehouse built using PostgreSQL.

## Layers
- **Raw Layer**: stores raw, unmodified data.
- **Staging Layer**: performs cleaning, validation, and standardization.
- **Dimensional Warehouse**: star schema composed of fact and dimension tables.
- **Data Marts**: analytics-ready aggregate tables.
